# 高考志愿顾问 Gaokao Advisor — 项目规则

## 环境
- **Python**: `/d/software/system/miniconda/python`（不要用系统 python）
- **端口**: 8083（8082 被 worldcup-predictor 占用）
- **工作目录**: `D:\gaokao-advisor\backend`（启动 uvicorn 时）

## 启动
```bash
/d/software/system/miniconda/python -m uvicorn app.main:app --host 0.0.0.0 --port 8083
```
启动前必须 `netstat -ano | grep 8083` 确认端口空闲。

## 关键规则
1. **先诊断再动手** — 每次工作前确认 Python 路径、端口、数据库状态
2. **启动前检查端口** — 不要先启动再查错
3. **中文缩写陷阱** — "物化生"匹配用单字 `"化"` `"生"`，不是 `"化学"` `"生物"`
4. **修bug加日志** — 不要反复重启猜测，先加 logging 查根因
5. **改代码后清缓存** — `find . -name __pycache__ -exec rm -rf {} +` 再重启

## 数据库
- 路径: `D:\gaokao-advisor\data\gaokao.db`
- 种子: `/d/software/system/miniconda/python data/seed_sichuan.py` (在 `D:\gaokao-advisor` 目录)
